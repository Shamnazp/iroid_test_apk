# Flutter Machine Test - iROID Technologies

## Overview
This project was developed as part of the Flutter Machine Test for iROID Technologies.

The application implements the given UI design and integrates the provided APIs while maintaining clean architecture and proper state management.

---

## Features
- UI implementation based on the provided Figma design
- API integration using REST services
- State management using Provider
- Clean and maintainable folder structure
- Proper loading and error handling
- Reusable widgets and clean code practices

---

## State Management
Provider is used for managing application state efficiently and maintaining separation between UI and business logic.
